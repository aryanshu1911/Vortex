# Vortex - Offensive & Defensive Network Scanner

Vortex is a professional CLI network scanner inspired by Nmap & Metasploit.  
Scan hosts or entire networks, grab banners, and identify common vulnerabilities.

## Features

- Interactive CLI (no command-line arguments required)
- Network and single host scanning
- Port scanning, banner grabbing
- Vulnerability detection (HTTP, SMB, SSH, etc.)
- JSON and TXT report generation
- Colorized, portfolio-ready output

## Installation

```bash
git clone https://github.com/yourusername/vortex.git
cd vortex
pip install -r requirements.txt
```

## Usage

```bash
python main.py
```
